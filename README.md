#  Python для анализа данных

## О курсе

Этот курс представляет собой практическое погружение в мир анализа данных с использованием Python. Вы пройдете путь от основ языка и SQL-запросов до построения и оценки предиктивных моделей. Основной упор сделан на инструментарий, который ежедневно используют бизнес-аналитики, data-аналитики и специалисты по Data Science для извлечения ценных инсайтов из данных.

## Структура курса

Курс разделен на 4 ключевые темы, каждая из которых включает практические и лабораторные работы.

### **Тема 1. Инструментарий аналитика: SQL и Pandas (10 баллов)**
*   **Практическая работа 1.1** (Тест) Написание SQL-запросов для извлечения данных. *(5 баллов)*
*   **Лабораторная работа 1.1** Комплексная обработка данных из БД с помощью Pandas. *(5 баллов)*

### **Тема 2. Исследовательский анализ (EDA) и визуализация бизнес-данных (10 баллов)**
*   **Практическая работа 2.1** (Тест) Выбор правильного типа визуализации под бизнес-вопрос. *(5 баллов)*
*   **Лабораторная работа 2.1** Проведение полного цикла EDA и создание аналитического отчета с гипотезами. *(5 баллов)*

### **Тема 3. Анализ временных рядов и подготовка признаков (10 баллов)**
*   **Практическая работа 3.1** (Тест) Подготовка данных для моделирования (кодирование, масштабирование). *(5 баллов)*
*   **Лабораторная работа 3.1** Анализ и визуализация временного ряда бизнес-показателей. *(5 баллов)*

### **Тема 4. Построение и оценка предиктивных моделей и итоговый контроль (70 баллов)**
*   **Практическая работа 4.1** (Тест) Оценка и сравнение качества различных моделей. *(5 баллов)*
*   **Лабораторная работа 4.1** Построение и оценка модели для прогнозирования оттока клиентов. *(5 баллов)*
*   **Итоговый тест (Зачет)** Комплексная проверка знаний по всему курсу. *(60 баллов)*

## Порядок выполнения работ

Все учебные материалы, лекции и задания находятся на портале Moodle.

*   **Практические работы** выполняются в виде тестов непосредственно на [портале преподавателя (Moodle)](http://95.131.149.21/moodle/course/view.php?id=43).

*   **Лабораторные работы** представлены в виде рабочих тетрадей Jupyter Notebook (`.ipynb`).
    1.  Ознакомьтесь с описанием и материалами к лабораторной работе по ссылкам ниже.
    2.  Выполните ваш индивидуальный вариант в файле `.ipynb`.
    3.  Создайте публичный репозиторий на **GitHub** или **GitVerse**.
    4.  Загрузите в него ваш итоговый файл `.ipynb` с выполненным заданием и всеми выводами ячеек.
    5.  Прикрепите ссылку на этот репозиторий в соответствующем задании на портале Moodle.

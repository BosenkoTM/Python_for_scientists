# Лабораторная работа 4.1. Построение и оценка модели для прогнозирования оттока клиентов

### Цель работы
Научиться применять модели машинного обучения (классификацию и регрессию) для решения реальных бизнес-задач, интерпретировать полученные результаты (метрики качества, важность признаков) и формировать практические рекомендации для бизнеса по удержанию клиентов.

### Инструкция по выполнению
1.  Используйте файл рабочей тетради **`bi_ml_4_lw_prepod.ipynb`** и набор данных **`Telco-Customer-Churn.csv`**.
2.  Выполните последовательно все задания (с 1 по 14), включая предобработку данных, обучение моделей и сегментацию.
3.  В разделе **"Часть 7: Индивидуальный проект"** выберите вариант задания из списка, найдите соответствующий датасет и выполните полный цикл анализа и моделирования.
4.  Используйте пайплайны (`Pipeline`) и трансформеры (`ColumnTransformer`) для обработки данных.
5.  Результат выполнения каждого пункта выведите на экран (метрики, графики, матрицы ошибок) и напишите текстовые выводы.

### Требования к сдаче работы
1.  Создайте публичный репозиторий на **GitHub** или **GitVerse**.
2.  Загрузите в него ваш Jupyter Notebook (`.ipynb`) с выполненными заданиями (основная часть + индивидуальный проект).
    *   **Важно:** убедитесь, что в файле сохранены все результаты выполнения ячеек (выводы кода и графики).
3.  Скопируйте ссылку на ваш репозиторий.
4.  Прикрепите эту ссылку в поле для ответа на задание на платформе **Moodle**.

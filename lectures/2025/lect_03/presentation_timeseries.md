# –ú–æ–¥–µ–ª—å ARIMA

**ARIMA (AutoRegressive Integrated Moving Average)** ‚Äî –∫–æ–º–±–∏–Ω–∏—Ä–æ–≤–∞–Ω–Ω–∞—è –º–æ–¥–µ–ª—å –¥–ª—è –ø—Ä–æ–≥–Ω–æ–∑–∏—Ä–æ–≤–∞–Ω–∏—è.

**–ö–æ–º–ø–æ–Ω–µ–Ω—Ç—ã ARIMA(p, d, q):**

| –ü–∞—Ä–∞–º–µ—Ç—Ä | –ó–Ω–∞—á–µ–Ω–∏–µ | –û–ø–∏—Å–∞–Ω–∏–µ |
|----------|----------|----------|
| **p** | AR(p) | –ü–æ—Ä—è–¥–æ–∫ –∞–≤—Ç–æ—Ä–µ–≥—Ä–µ—Å—Å–∏–∏ |
| **d** | I(d) | –ü–æ—Ä—è–¥–æ–∫ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏—è |
| **q** | MA(q) | –ü–æ—Ä—è–¥–æ–∫ —Å–∫–æ–ª—å–∑—è—â–µ–≥–æ —Å—Ä–µ–¥–Ω–µ–≥–æ |

**AR:** Y(t) = c + œÜ‚ÇÅY(t-1) + œÜ‚ÇÇY(t-2) + ... + Œµ(t)
**MA:** Y(t) = c + Œ∏‚ÇÅŒµ(t-1) + Œ∏‚ÇÇŒµ(t-2) + ... + Œµ(t)

# –í—ã–±–æ—Ä –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ ARIMA

**–ê–ª–≥–æ—Ä–∏—Ç–º –≤—ã–±–æ—Ä–∞ –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤:**

1. **–ü–∞—Ä–∞–º–µ—Ç—Ä d:**
   - –ü—Ä–∏–º–µ–Ω—è—Ç—å –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏–µ –¥–æ –¥–æ—Å—Ç–∏–∂–µ–Ω–∏—è —Å—Ç–∞—Ü–∏–æ–Ω–∞—Ä–Ω–æ—Å—Ç–∏
   - –ü—Ä–æ–≤–µ—Ä—è—Ç—å ADF —Ç–µ—Å—Ç –ø–æ—Å–ª–µ –∫–∞–∂–¥–æ–≥–æ –¥–∏—Ñ—Ñ–µ—Ä–µ–Ω—Ü–∏—Ä–æ–≤–∞–Ω–∏—è

2. **–ü–∞—Ä–∞–º–µ—Ç—Ä p (–∏–∑ PACF):**
   - –ù–∞–π—Ç–∏ –ª–∞–≥, –ø–æ—Å–ª–µ –∫–æ—Ç–æ—Ä–æ–≥–æ PACF –æ–±—Ä—ã–≤–∞–µ—Ç—Å—è

3. **–ü–∞—Ä–∞–º–µ—Ç—Ä q (–∏–∑ ACF):**
   - –ù–∞–π—Ç–∏ –ª–∞–≥, –ø–æ—Å–ª–µ –∫–æ—Ç–æ—Ä–æ–≥–æ ACF –æ–±—Ä—ã–≤–∞–µ—Ç—Å—è

4. **–ö—Ä–∏—Ç–µ—Ä–∏–∏ –∫–∞—á–µ—Å—Ç–≤–∞:**
   - AIC (Akaike Information Criterion) ‚Üí –º–µ–Ω—å—à–µ = –ª—É—á—à–µ
   - BIC (Bayesian Information Criterion) ‚Üí –º–µ–Ω—å—à–µ = –ª—É—á—à–µ
